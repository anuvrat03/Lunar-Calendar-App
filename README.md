# 🌙 Interactive Lunar & Hindu Calendar

An interactive web-based calendar built with **p5.js** that calculates and visualizes lunar data dynamically. When a user selects any date, the application computes the precise moon phase, determines the corresponding Hindu calendar attributes (*Paksha* and *Tithi*), and provides a countdown to upcoming major lunar events.

## ✨ Features

* **Interactive Grid Layout:** Seamlessly navigate through months and years using intuitive UI controls.
* **Vector Moon Rendering:** Real-time mathematical simulation drawing the exact lighting curve (crescent to gibbous) of the moon.
* **Vedic Astrology Integration:** Automatically maps the astronomical synodic lunar period into:
  * **Paksha:** Shukla Paksha (Waxing / Bright half) or Krishna Paksha (Waning / Dark half).
  * **Tithi:** Maps the specific lunar day (e.g., Prathama, Ashtami, Ekadashi, Amavasya, Purnima).
* **Lunar Event Countdown:** Live dynamic feedback calculating exactly how many days remain until the next Full Moon (*Purnima*) and New Moon (*Amavasya*).
